
## configuration used

### Model confs


*Syntax used*

```
<<access_intended>><<type_of_access>>_<<modelsname>>

--- access_intended 
    F - Front_END
    B - Back_END
    I - Intermediate_END

--- type_of_access
    '' - full
    C - create
    R - Read
    U - update
    D - Delete

--- modelsname
```
